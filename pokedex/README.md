## 🚀 Getting started

Clone the project and access the folder.

```bash
git clone https://github.com/jeffymesquita/teste-frontend-imobpower.git
cd pokedex
```



Follow the steps below:

```bash
# Install the dependencies
$ yarn install
# Start the project
$ yarn start
```
